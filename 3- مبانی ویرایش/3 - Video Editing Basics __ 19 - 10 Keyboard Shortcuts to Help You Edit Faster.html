Here are 10 shortcuts to help you speed up your workflow. I'll be sharing a few more throughout the course. These are ones to get familiar with right away!

Note: Command is for Mac users, and Control (CTRL) is for PC users. I use a Mac, so whenever I say 'command,' use control if you're on a PC. Also, on a Mac we use Option. On a PC it is ALT.

  • Import: CTRL + I
  • Rewind, Pause, Play: J, K, L
  • Increase/Decrease Clip Volume: [ or ]
  • Copy Clips: OPTION-click a clip, drag to a new part of the timeline
  • New Folder/Bin in Project Panel: CTRL + B
  • Nudge Clip: (SHIFT) + ALT + LEFT / RIGHT
  • Razor Blade Editor: C
  • Link / Unlink: CTRL + L
  • Mark In / Out: I / O
  • Export Media: CTRL + M
